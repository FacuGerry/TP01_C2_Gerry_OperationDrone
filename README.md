# TP07_C2_Gerry_OperationDrone

[EN] Welcome, soldier! Your mission is to operate a drone and erradicate all the nefarious criminals around this city. Beware! You mustn't hurt any citizens, thus we'll take out some points from your score.

[ES] ¡Bienvenido, soldado! Tu misión es operar un dron y erradicar a todos los nefastos criminales que deambulan por la ciudad. Cuidado! No debes herir a ningún civil, te quitaremos puntos de ser así.

Controls / Controles:

* WASD -> Movement / Movimiento
* Space / Espacio -> Go up / Ir hacia arriba
* Left Ctrl / Control izq -> Go down / Ir hacia abajo
* Left click / Click izq -> Shoot / Disparar
* F -> Change camera / Cambiar cámara
* Q -> Secondary shoot / Disparo secundario
* T -> Toogle ray for aiming / Encender rayo para apuntar
* Cursor -> Look around / Mirar
